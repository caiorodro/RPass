
Este software é desenvolvido na plataforma UWP (Universal Windows Platform) para até então windows 10 e tem o 
objetivo de guardar senhas referenciadas.

Granvando as senhas em database local SQLite, não há limite de gravação de senhas e tem possibilidade de anexar 
arquivos de qualquer tipo em cada senha gravada.

O sistema está escrito n linguagem é C#. Divirta-se!
